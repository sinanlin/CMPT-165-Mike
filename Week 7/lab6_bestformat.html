Part 2: file size and comment

For the first picture, I would prefer to pick JPEG as the better file format because the file size is ralatively small so that the webpage can be loaded faster. In the meantime, the image quality does not need to be compromised.

For the second picture, I would either use GIF or PNG because the JPEG does not support transparency. GIF or PNG in this case both have ralativly small file size. The image quality is also acceptable.
